# forge-server-jars-for-old-mc
This is the repository of the OLDEST Forge ServerJars.

# How i made it?
This is all you need to make a Server jar from 1.0-1.5.1 Forge 'universal' package:

- HxD Editor
- Java Decompiler (JD)
- Server jar for the version you want to make
- 'universal' package from minecraftforge.net
- [OPTIONAL BUT COMMON] JARs of libraries for Forge from Maven or OW2 since the original ones may be down
